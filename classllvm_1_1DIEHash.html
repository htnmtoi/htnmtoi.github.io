uint64_t DIEHash::computeCUSignature ( const DIE &  Die)

Computes the CU signature.

This is based on the type signature computation given in section 7.27 of the DWARF4 standard.

It is an md5 hash of the flattened description of the DIE with the inclusion of the full CU and all top level CU entities.

uint64_t DIEHash::computeDIEODRSignature ( const DIE &  Die)

Computes the ODR signature.

This is based on the type signature computation given in section 7.27 of the DWARF4 standard.

It is the md5 hash of a flattened description of the DIE with the exception that we are hashing only the context and the name of the type.
