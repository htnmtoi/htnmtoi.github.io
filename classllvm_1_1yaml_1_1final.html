Detailed Description

A null value.

Represents an alias to a Node with an anchor.

Represents a YAML sequence created from either a block sequence for a flow sequence.

Represents a YAML map created from either a block map for a flow map.

A key and value pair.

A block scalar node is an opaque datum that can be presented as a series of zero or more Unicode scalar values.

A scalar node is an opaque datum that can be presented as a series of zero or more Unicode scalar values.

Example: !!null null

Example: Adena

Example: | Hello World

While not technically a Node under the YAML representation graph, it is easier to treat them this way.
