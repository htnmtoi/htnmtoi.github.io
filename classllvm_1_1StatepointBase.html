Detailed Description

template<typename InstructionTy, typename ValueTy, typename CallSiteTy>
class llvm::StatepointBase< InstructionTy, ValueTy, CallSiteTy >

Analogous to CallSiteBase, this provides most of the actual functionality for Statepoint and ImmutableStatepoint.

It is templatized to allow easily specializing of const and non-const concrete subtypes. This is structured analogous to CallSite rather than the IntrinsicInst.h helpers since we want to support invokable statepoints in the near future. TODO: This does not currently allow the if(Statepoint S = ...) idiom used with CallSites. Consider refactoring to support.
