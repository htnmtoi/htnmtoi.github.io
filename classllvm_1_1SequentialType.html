Detailed Description

SequentialType - This is the superclass of the array, pointer and vector type classes.

All of these represent "arrays" in memory. The array type represents a specifically sized array, pointer types are unsized/unknown size arrays, vector types represent specifically sized arrays that allow for use of SIMD instructions. SequentialType holds the common features of all, which stem from the fact that all three lay their components out in memory identically.
