Detailed Description

This class stores information about jump-instruction tables created by the JumpInstrTables pass (in lib/CodeGen/JumpInstrTables.cpp).

Each table is a map from a function type to a vector of pairs. The first element of each pair is the function that has the jumptable annotation. The second element is a function that was declared by JumpInstrTables and used to replace all address-taking sites for the original function.

The information in this pass is used in AsmPrinter (lib/CodeGen/AsmPrinter/AsmPrinter.cpp) to generate the required assembly for the jump-instruction tables.
